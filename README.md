## II. ARCHITECTURE DIRECTORY (COMPACT BLUEPRINT v4.0)

### 1. Vocal Register (Pitch) #P.#
* `#Pw#`: Whistle Register (ultra-high)
* `#Pu#`: Up one register
* `#Pn#`: Normalized
* `#Pd#`: Down one register
* `#Ps#`: Sub-Harmonic Rumble (ultra-low)

### 2. Backing Vocals & Harmony #BV.#
* `#Bm#`: Male harmony
* `#Bf#`: Female harmony
* `#Bc#`: General choir
* `#Bm_c#`: Male choir
* `#Bf_c#`: Female choir

### 3. Continuity & Structural Stops #()K+/-#
* `#()K+#`: Open phrase / Weave (Keep vocal channel active)
* `#K-#`: Anchor Stop (Hard cut / Close segment)
